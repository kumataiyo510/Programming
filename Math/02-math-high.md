# 特別授業１（幾何の最終兵器ベクトル）
ベクトルは幾何の問題をいったん代数に置き換えて解く。幾何の場合は補助線を引いたりして図形的に解決を図ってきたが、代数にしてしまえば機械的に式を触ることで問題を解くことができるようになる利点が出る。
## ベクトルの概念を理解する
ベクトルとは「大きさ」と「向き」という二つの異なるデータを格納できる入れ物のことで、一般化すると二つの情報を格納できる入れ物である、その他一般的なものはスカラーといい一つの情報しか含まれていない。一方ベクトル以上に、無数のスカラーを格納することができる概念としてテンソルも存在する。
## なぜベクトルが生まれたか
図形を代数の世界に持ち込みたかったが、「大きさ」と「向き」をスカラーの考え方では一つの式に表せなかった。それをベクトルという二つのスカラーを持つ入れ物を発明することで可能になった。
## ベクトルの書き方
**a** 太字にする（Vector a）<br>

# 特別授業２（微分積分）

## 三角形の面積
底辺と高さが1の直角二等辺三角形の面積はいくらか？（一次関数y=xで切り落としている）<br>
◢　1 * 1 / 2 = 1 / 2<br>
次に、斜辺が放物線になっていた場合の面積は？<br>
放物線はy = x^2になっており、この場合の面積は<br>
正方形の面積 / 3 になるので、1 / 3が答えになる。<br><br>
ここまでの情報から、辺の長さが１の正方形において、y=xの一次関数で面積を切り落とせば1/2、y=x^2の二次関数で切り落とせば1/3になることが分かった。実はこれは３次以降の関数でも法則が適用され続ける。この1と1/2、2と1/3、3と1/4といった関係積が「積分」になる。<br>

## 微積分の考え
ニュートン「三角形の面積を調べるなら、対象物を細い縦上の短冊に等間隔にスライスして、それぞれの短冊の面積を求め、最後に足しましょう。三角の場合短冊にすると、先の三角の部分が残るが、細々にスライスすれば面積は小さくなるから無視すればよい」これが微分の考えの始まり。<br>
```
三角形の面積を求める
◢　において等辺をaとすると面積はa^2 / 2となる。
この三角形を一度座標軸の中に埋め込んでみる。
斜辺がy=x、高さがx=aの縦線で表される。
まずは、３等分にスライスしてみる。
x軸は0-a/3、a/3-2a/3、2a/3-aとなる。
長方形を作るため、それぞれ高さを求める。
作れない(0-a/3の時)、a/3(a/3-2a/3の時)、2a/3(2a/3-aの時)となる。
次にそれぞれの面積を求めて、足してあげる。この分けたものを足していくことを「積分」という。
a/3 * a/3 + a/3 * 2a/3
= a^2 / 9 + 2a^2 / 9
= 3a^2 / 9
= a^2 / 3
正方形の1/3の面積になる。
分割が大きかったので捨てた部分の誤差が大きくなってしまった。ニュートンは無限大に分割したらよいと考えた。(n等分)

長さをaとすると、底辺の長さはa/nで(n-1)a/n分割される。
面積はy=xの関数なので高さはxと同じになるということで。
a/n * a/n + a/n * 2a/n + a/n * 3a/n +...+a/n * (n-1)a/n
= a/n{a/n + 2a/n + 3a/n +...+(n-1)a/n}
= a^2/n^2{1 + 2 + 3 +...+(n-1)}
{}の中が公差１の等差数列になっているので、
= a^2/n^2 × n(n-1)/2
nは無限大に近い数を想定しており、1を引いたところで誤差の範囲であるならば無視をして(n-1)はnと考えよう。
= a^2/n^2 × n^2/2
= a^2 / 2
∴三角形の面積は四角形の面積(a^2)に1/2をかけたものになる。
```
